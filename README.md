# WEATHERLY

WEATHERLY is a static, client-side weather app built with HTML, CSS, and vanilla JavaScript.

## Features

- Search current weather by city and country
- Enter your OpenWeatherMap API key directly in the page
- View temperature, condition, humidity, wind speed, and weather icon
- Responsive minimalist layout for mobile, tablet, and desktop
- Friendly inline handling for empty fields, invalid locations, invalid API keys, and network issues

## Run Locally

1. Open `index.html` in a browser.
2. Enter a valid OpenWeatherMap API key.
3. Enter a city and country, then select `Get Weather`.

## API

This app uses the OpenWeatherMap Current Weather endpoint:

`https://api.openweathermap.org/data/2.5/weather`
